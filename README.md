<h3 align="center" style="margin-top:0;">✨ What is Tetris?</h3>
Tetris is a world-famous block puzzle game created in 1984 by Soviet programmer Alexey Pajitnov.
The name "Tetris" comes from Greek word <code>tetra</code> (meaning four, every block consists of 4 small squares) plus tennis, which is the creator’s favorite sport.

Stack falling geometric blocks, fill full horizontal lines to clear them, earn scores and level up continuously!

## 🎮 Complete Game Rules & Operation Guide
<div style="background:linear-gradient(135deg,#84fab0 0%, #8fd3f4 40%, #d299c2 100%);padding:24px;border-radius:18px;">
<h4>Basic Gameplay Rules</h4>
1. Random tetromino shapes fall down from the top of the game board continuously
2. Move left/right, rotate and drop blocks to fill blank space neatly
3. Once a horizontal row is fully filled without gaps, it will be eliminated automatically and reward scores
4. Clearing multiple lines at the same time gets extra high score bonus
5. The falling speed increases as your level rises
6. Game Over when stacked blocks reach the top edge of the game area

<h4>Keyboard Controls</h4>
- ← Left Arrow: Move block left
- → Right Arrow: Move block right
- ↓ Down Arrow: Soft drop (speed up falling)
- ↑ Up Arrow: Rotate block shape
- Space Bar: Hard drop (instantly fall to bottom)
- P Key: Pause / Resume game
</div>
